Project Experience

  • 华为 - 代码质量审查项目（500 RMB）

    基于我们 ASE’20 关于 API 误用检测的研究，我们开发了基于知识图谱的 API 误用检测工具。通过 API 官方文档构建 API 使用知识图谱，用以检测代码中潜在的 AIP 误用。这项研究被应用到华为的代码质量审查项目中。

  • 华为 - 基于分布式应用全息知识图谱的故障根因定位模型（900 HKD）

    华为云 Stack 会托管部署用户的大量分布式应用，涉及的云服务范围广泛。一旦单个应用出现问题可能引发连锁反应，需要快速定位根因节点。本项目通过构建全息知识图谱，将云系统中应用、中间件、底层资源进行整合，并结合整个拓扑图中的各层级节点的告警、响应时延、错误率等多种指标，综合评估某个节点成为当前故障根因节点的概率，从而保证较高的准确率。
